# Standard Template Library

## There are three main elements in STL
--> Container
--> Iterator
--> Algorithm 

## Type of Containers

--> Sequence Container 
	array, vector, list, forword_list, deque.

--> Associative Container
	set, multiset, map, multimap.

--> Container Adaptor
	stack, queue, priority queue.

## Type of Iterator

--> Input
--> Output
--> Forword
--> Bi-directional
--> Random access
